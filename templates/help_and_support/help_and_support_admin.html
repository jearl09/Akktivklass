Creating a New Class

Follow these steps to create a new class as an Admin:

  1. Go to the Admin Panel.
  2. Select "Classes" from the sidebar menu.
  3. Click the "Create New Class" button.
  4. Fill in the required class information such as name, code, and assigned teacher.
  5. Click "Save" to create the class.

Managing Attendance

  • Navigate to the "Attendance" tab in your class.
  • Mark present, absent, or late with one click.
  • Use filters to view attendance history.
  • Export reports in CSV or PDF formats.

Account Security

Best practices for keeping your AktivKlass account secure:

  • Use a strong, unique password and change it regularly
  • Enable two-factor authentication in your account settings
  • Never share your login credentials with others
  • Always log out when using shared computers
  • Review active sessions in your account settings periodically

Class Management

Comprehensive guide to managing your classes:

  • Edit class details anytime from the class settings
  • Archive inactive classes to keep your dashboard organized
  • Use the student management section to add/remove students
  • Assign co-teachers to share class management responsibilities
  • Set custom permissions for different user roles

Setting Up Your Account

  1. Complete your profile information after registration
  2. Verify your email address
  3. Set up two-factor authentication for security
  4. Configure your notification preferences
  5. Upload a profile picture for easy identification

Creating Your First Class

  1. Navigate to the Classes section in your dashboard
  2. Click "Create New Class"
  3. Enter basic class information (name, subject, grade level)
  4. Set your class schedule and meeting times
  5. Configure default attendance settings
  6. Click "Create Class" to finish

Managing Students

  • Add students individually or import from a CSV file
  • Organize students into groups for easier management
  • Set custom student identifiers if needed
  • Monitor student activity and progress
  • Communicate with students through the messaging system
  • Archive students when they leave your class

Effective Class Management

  • Establish clear naming conventions for your classes
  • Use color coding to visually organize different subjects
  • Set up recurring activities to save time
  • Utilize co-teachers for collaborative classes
  • Regularly review and archive inactive classes
  • Take advantage of templates for similar classes

Attendance Tracking Tips

  • Take attendance at the same time each day to establish routine
  • Use keyboard shortcuts for faster marking
  • Set up automatic reminders for attendance taking
  • Review attendance patterns weekly to spot trends
  • Configure custom attendance codes for your institution
  • Export attendance data regularly for record keeping

Security Recommendations

  • Require all users to enable two-factor authentication
  • Regularly review and update user permissions
  • Conduct periodic security training for staff
  • Monitor login attempts and suspicious activity
  • Establish a password policy for all users
  • Keep student data confidential and share only when necessary
